(inner) calcular() → {void}

Esta función captura dos valores numéricos y una operación seleccionada desde un formulario HTML. Realiza el cálculo correspondiente y muestra el resultado en un elemento específico de la página. Operaciones soportadas: - Suma (+) - Resta (-) - Multiplicación (×) - División (÷) - Módulo (%) - Potencia (^) - calculada con bucle sin usar ** Características: - Validación de entrada numérica - Manejo de errores para división por cero - Cálculo de potencia mediante bucle - Soporte para exponentes negativos
